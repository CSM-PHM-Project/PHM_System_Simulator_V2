SPEARS

SPEARS is an open-source project developed by the Van Bossuyt Research Lab at the Colorado School of Mines and is intended for use as a reseach tool. SPEARS can be used to develop and test autonomous behaviors in highly customizable environments. See the links on the left for more details.

SPEARS requires source code editing to be use meaningfully but you can check out the project by downloading it here. SPEARS requires Java 8 to run which can be downloaded here.
